-----------------------------------------------------------------------------------------
-	-> This directory contains MATLAB code for the summer internship projet:	-
-	Mesure statistique de l'apparition de phénomènes de propagation anormale 	-
-	entre un émetteur constant et un récepteur basé à Rambouillet			-
-----------------------------------------------------------------------------------------
-	-> MATLAB version used:								-
-		- R2012b (8.0.0.783) 64-bit						-
-					 						-
-	-> Special toolboxes used: 							-
-		- Statistics Toolbox (Version 8.1)					-
-----------------------------------------------------------------------------------------
-	-> Reading and writing of data:							-
-											-
-	- readMeasures.m								-
-		Read data from a file generated by the LabView application		-
-											-
-	- readAllMeasures.m								-
-		Read data from multiples files generated by the LabView application	-
-											-
-	- saveData.m									-
-		reads "data" and saves its content at "directory" in a .txt file called -
-		"name"									-
-											-
-	- myCell2Mat.m									-
-		Creates array of numbers from cell of strings representing 		-
-		numbers (either with decimal point or comma)				-
-----------------------------------------------------------------------------------------
-	-> Selection of desired data:							-
-											-
-	- filterDate.m									-
-		Filter desired dates from data structure				-
-											-
-	- filterFile.m									-
-		Read "inFile" data, divide into small sets of "timeScale" size,		-
-   		tries to find the probable real data and saves new data in "outFile" 	-
-											-
-	- filterAllFiles.m								-
-		Applies "filterFile.m" to a set of desired files at "pathIn" and saves	-
-		the output files at "pathOut"						-
-											-
-	- findSeries.m									-
-		Tries to detect the real direct bursts on "data", according to		-
-		the theoretical antenna turning time of "timeGap" and taking a into	-
-		account a deviation of "timeDeviation", saving the results in the	-
-   		"indexes" matrix							-
-											-
-	- getBiggerValues.m								-
-		Gets bigger value compared to its neighboring values 			-
-		(MUST ONLY BE USED AFTER FINDING THE REAL SERIES)			-
-											-
-	- removeNoiseFromFile.m								-
-		Select only data with power over "limit"				-
-											-
-	- removeNoiseFromAllFiles.m							-
-		Loads files from "inPath", select only data having above "limit" and 	-
-		saves files at "outPath"						-
-											-
-	- testData.m									-
-		Tries to detect the real direct bursts on "data", accoarding to the	-
-		the theoretical antenna turning time of "timeDif" and saves the indexes -
-   		indexes of the results in "indexes" matrix				-
-		[NO LONGER IN USE - DEPRECATED IN FAVOR OF findSeries.m]		-
-											-
-----------------------------------------------------------------------------------------
-	-> Data structure handling functions:						-
-											-
-	- selectData.m									-
-		Select data from "data" array using a selection vector			-
-											-
-	- arrangeData.m									-
-		Arrange data into block for "separateData.m" function. Takes from 	-
-		"data" array and select only data between "h1" and "h2" indexes		-
-											-
-	- separateData.m								-
-		Separate data into blocks accoarding to desired time window		-
-											-								
-	- joinData.m									-
-		Select data from "data" array using a selection vector			-
-----------------------------------------------------------------------------------------
-	-> Other Files:									-
-											-
-	- systemLinkBudget.m								-
-		Simple script that loads all the chracteristic values of the system	-
-											-
-	- systemCharacteristics.txt							-
-		Keeps system characteristics (data order MUST NOT BE CHANGED)		-
-											-
-	- getCDF.m									-
-		Calculates CDF from data						-
-		[NO LONGER IN USE - DEPRECATED IN FAVOR OF STATISTICS TOOLBOX]		-
